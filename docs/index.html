vplay

Run videos on multiple screens or rooms with independent control from one easy, fast, lightweight terminal UI.

View on GitHub Download latest beta Discussions

Install

brew tap rztrace/vplay https://github.com/rztrace/vplay
brew install vplay
vplay

Portable macOS builds are available from GitHub Releases.

Why It Exists

  • Most players are built around one video window.
  • Multi-output tools are often heavy, app-centric, or signage-oriented.
  • vplay stays local, fast, keyboard-first, and focused on stable operation.
  • Each active screen or room gets its own playback session and controls.

Use Cases

  • Private screening rooms, video booths, galleries, studios, and installations.
  • Several TVs, projectors, rooms, or screens controlled from one machine.
  • Side-by-side references, loops, playlists, and clipped sections.
  • People who need a lightweight TUI instead of a full control system.
