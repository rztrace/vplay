vplay

A macOS terminal video player for controlling local videos, playlists, clips, downloads, and multiple displays from one TUI.

View on GitHub Download latest beta Discussions

Install

brew tap rztrace/vplay https://github.com/rztrace/vplay
brew install vplay
vplay

Portable macOS builds are available from GitHub Releases.

Why Use It

  • Control mpv from a keyboard-first terminal UI.
  • Play different videos on different displays at the same time.
  • Keep playlists, renamed files, clipped portions, and download queues close to the media folder.
  • Extend the TUI with system and user modules.

Good Fit

  • Multi-monitor Mac setups.
  • Local video collections.
  • Fast playlist workflows.
  • People who prefer TUI tools over media-center apps.
